Pachyderm File System

The Pachyderm File System (pfs) is a copy-on-write distributed file system that offers complete version control for your data. You can commit, branch, and merge huge data sets, just like you’ve always done with code. If something goes wrong, you can instantly revert back to a previously known good state. Learn more →

Pachyderm Pipeline System

The Pachyderm Pipeline System (pps) lets you define analysis pipelines, chain them together, and run them efficiently. Pipelines of jobs are defined as Docker containers, so you can use any language and any libraries you want. If it fits in a container, you can use it for data analysis. Learn more →
