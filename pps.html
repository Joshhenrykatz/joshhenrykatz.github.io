Pachyderm Pipeline System (pps)

Pps is a containerized analytics framework.

Pipelines

Rather than thinking in terms of map or reduce jobs, pps thinks in terms of pipelines expressed within a container. A pipeline is a generic way expressing computation over large datasets and it’s containerized to make it easily portable, isolated, and easy to monitor.

Developer-driven infrastructure

In Pachyderm, all analysis runs in containers. You can write them in any language you want and include any libraries. Instead of needing an ops team to manage dependencies in the cluster, the developers themselves have complete control over the runtime environment for their analysis. Test the analysis locally and then seamlessly run it in the cluster.

Smart DAG computation

Similar to as Spark does with RDDs, Pachyderm uses its commit-based file system to efficiently track computations. Because it always knows the results of past computations, Pachyderm can heavily optimize analysis by only computing the diff. This makes it ideal for iterative algorithms such as PageRank.

Batched and streaming analytics in one system

In Pachyderm, you can seamlessly turn a batched job into a streaming job by increasing the frequency of commits (snapshots). Commits are insanely cheap so you can make one every second if you want! Pachyderm knows exactly what data has changed in each commit and recomputes only the diff to give you real-time results.

Analyze data directly out of S3

Migrating data is annoying. So don’t! You can run pps analytics pipelines directly on top of S3. Just point it to your S3 bucket and add your credientials (link). If reading directly out of S3 is too slow or expensive, you can store “hot” data in pfs, which also provides temporal history for your data and pipelines.

Prefabricated pipelines (coming soon!)

You wouldn’t make your own Redis deployment container when there’s an awesome one on Docker Hub already, would you? Now you get the same thing for your data analytics! Pachyderm and our developer community are creating point-and-click analysis pipelines so you can get started in minutes.
